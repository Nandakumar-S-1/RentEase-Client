# Tech Stack & Libraries

The RentEase Client uses a modern frontend stack to ensure performance, maintainability, and a premium user experience.

## 🧱 Core Technologies

- **[React](https://react.dev/)**: For building a component-based UI.
- **[Vite](https://vitejs.dev/)**: The next-generation frontend tool for lightning-fast builds and HMR.
- **[TypeScript](https://www.typescriptlang.org/)**: For type safety across the application.
- **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework for rapid and consistent styling.

## 🛠 Tools & Configuration

- **PostCSS**: For specialized CSS processing.
- **ESLint**: Configured with strict TypeScript rules to maintain code quality.
- **SWC/Babel**: For high-performance JavaScript/TypeScript compilation.

## 📁 Directory Structure (Planned)

```text
src/
├── assets/       # Images, webfonts, etc.
├── components/   # Reusable UI components
├── hooks/        # Custom React hooks
├── pages/        # Main application pages
├── services/     # API interaction logic
└── styles/       # Global CSS and Tailwind configuration
```
